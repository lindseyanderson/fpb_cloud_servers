Objective:
=====================
  To create a Boopstrap/Flask/Pyrax application to act as a simple interface to the Rackspace Cloud Server's API.
 
Inputs:
---------------------
  - Username
  - API-Key

Outputs:
---------------------
  - Username submitted
  - Errors

  - Cloud Server List
    - Cloud Server Name
    - Cloud Server ID
    - Public IPv4 Address
    - Public IPv6 Address
    - Private IPv4 address
